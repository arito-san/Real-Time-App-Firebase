## booMessage
This application is an app clone of Whatsapp made with Firebase (Back end) and React-Native (Front end) to talk to friends!

## Technology 

Here are the technologies used in this project.

* React-native  0.70.2
* JavaScript/Typescript
* LottieFiles
* React-navigation
* momentJs
* flash-message

## Services Used

* Github
* Firebase
